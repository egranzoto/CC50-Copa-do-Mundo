# Copa do Mundo - Simulação de Torneios FIFA

Projeto desenvolvido no curso CC50 (Harvard CS50) para simular torneios da Copa do Mundo utilizando Python e probabilidades baseadas no ranking FIFA.

## 📌 Sobre o Projeto

O programa realiza milhares de simulações de uma Copa do Mundo em formato mata-mata para estimar a probabilidade de cada seleção vencer o torneio.

As probabilidades são calculadas com base no rating FIFA de cada equipe.
